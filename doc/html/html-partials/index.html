Widely

Widely es una aplicacion web de libre distribucion bajo la licencia GPL que nace con la necesidad de centralizar el proceso de desarollo de una aplicacion.

En un ambiente ordinario un IDE es un marco de trabajo que soporta el desarrollo rapido de aplicaciones de un lenguaje de programacion en particular o, algunos casos, el diseño del IDE permite la adaptacion de multiples lenguajes por medio de modulos adicionales.

En un ambiente web un IDE es un centro de trabajo en donde el programador encuentra toda las herramientas necesarias para realizar su trabajo unicamente con el uso de un explorador web (Browser).

La centralizacion del proceso agrega nuevas funcionalidades no incluidas en un IDE estandar, estas son:

  1. Compilacion centralizada y distribuida:
    Es posible configurar el IDE de tal forma que la compilacion de codigo fuente se haga de forma centralizada (en un servidor) y que a la vez el servidor distribuya la compilacion a traves de la red a servidores dedicados.

  2. Control de Versiones Integrado:
    Con la centralizacion de la ubicacion de los archivos de un proyecto se abre la posibilidad de tener un repositorio local (realmente en el servidor) que facilete el control de las versiones de los archivos y que permita a multiples personas editar estos archivos simultaneamente.

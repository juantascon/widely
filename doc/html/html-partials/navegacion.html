Modelo de Navegacion

El diagrama de navegacion muestra el flujo de navegacion que un usuario final puede seguir, el siguiente es un diagrama de navegacion de aplicacion web modificado para la adaptacion a la tecnica de desarrollo AJAX

Este diagrama se creo utilizando la herramienta de libre distribucion Dia, puede ser visto en cualquier visor de imagenes

Acualmente se encuentra en dos formatos:

  1. En formato dia: navegacion.dia
  2. En formato png: navegacion.dia.png
